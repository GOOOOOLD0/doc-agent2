## 2. 主要内容摘要

1. 该法设立电信监察基金（Fundo de Fiscalização das Telecomunicações, FISTEL），并规定电信监察相关收入和费用。
2. 与电台许可直接相关的两项费用是安装监察费 TFI（Taxa de Fiscalização de Instalação）和运行监察费 TFF（Taxa de Fiscalização de Funcionamento）。
3. TFI 与电台许可证书的签发相关；TFF 为持续运行监管而按年缴纳。
4. 法律附件按电台类别给出 TFI 基础金额，其中包括卫星支持业务终端、不同口径地球站、移动地球站、GSO 空间站和 NGSO 系统。
5. 网页是跨多年修订的合并文本，显示多个历史税率或旧条文；实际应缴金额不能仅依据页面中任一可见数字确定。

## 3. 主要原始信息

### 3.1 TFI 和 TFF

- 第 6 条：TFI 在向电信服务及频率使用的被授权者等签发电台许可证书时发生。
- 第 6 条：TFF 用于对电台运行进行持续监管，按年征收。
- 第 8 条网页所示后续文本规定 TFF 通常在每年 3 月 31 日前缴纳，并显示为相应 TFI 的 33%；同一合并页面也保留早期 50% 或 45% 等历史文本。
- 经通知后逾期不缴可能导致相关授权或权利被宣告失效，具体程序需结合现行法核对。

### 3.2 附件中的卫星相关电台类别

网页合并文本所列 TFI 基础金额包括：

| 电台类别 | 页面所列金额（巴西雷亚尔） |
| --- | ---: |
| 全球卫星通信终端 | 26.83 |
| 受中心站控制、发射天线口径小于 2.4 米的小型地球站 | 201.12 |
| 数据网络等用途的中心地球站 | 402.24 |
| 发射天线口径大于 4.5 米的大型地球站 | 13,408.00 |
| 移动发射地球站 | 3,352.00 |
| GSO 空间站，按每颗卫星 | 26,816.00 |
| NGSO 空间站，按每个系统 | 26,816.00 |

上述金额是本来源页面附件中可见的法定表值，不等同于对任一具体项目的最终缴费结论。

### 3.3 与落地许可的关系

- 该法主要说明许可后的电台安装和持续监管收费基础。
- 它不替代卫星开发权公共价格、服务授权价格、频率使用价格或产品认证相关费用。
- 实际项目可能同时涉及多个电台类别和其他收费依据。

## 4. 关键原文依据

| 位置 | 原文要点 |

## 5. 信息不足或需人工复核

- 网页同时保留多个时期的修订文本，必须确认哪一比例和哪一附件金额在申请或缴费时仍有效。
- 需核对是否有后续法律规定减免、调整、暂停或改变 TFI/TFF 的征收方式。
- 页面金额未结合具体站点数量、设备类型、许可证书签发方式或 Anatel 缴费通知计算，不能作为最终报价。
- 该来源没有涵盖第 748/2021 号决议规定的卫星开发权公共价格，也未给出完整的在线缴费步骤。
- 正式财务判断应复核 DOU、Anatel 最新收费指南和具体账单。
