# 国家卫星落地许可证据映射模板

## 使用规则

本文件仅定义 Evidence Matrix 的格式，不保存任何国家事实。

Hermes 在执行 `build`、`update` 或 `full` 时：

- 输入：
  - `wiki/raw/landing_rights/<country>/source_inventory.md`
  - `wiki/raw/landing_rights/<country>/source_notes/`
- 输出：
  - `wiki/raw/landing_rights/<country>/evidence_matrix.md`
- 不得修改本模板；
- 不得使用其他国家案例补充目标国家事实；
- 必须逐份读取实际存在的 Source Notes；
- 一个证据项只表达一个主要结论；
- 同一 Source Note 可以支持多个证据项；
- 每个证据项必须说明“能够支持”和“不能支持”；
- 不得使用宽表格保存全部证据项；
- 不得按“一份 Source Note 一行”的方式生成；
- Evidence Matrix 未通过检查前，不得生成 `00-09` 案例。

生成目标国家文件时，将以下占位符替换为实际内容：

- `<Country English Name>`：国家英文名称；
- `<country>`：英文小写下划线形式；
- `<国家>`：国家中文名称；
- `<source_note_id>`：实际 Source Note 文件名，不含 `.md`。


### 输出净化

生成的 `evidence_matrix.md` 必须是纯 Markdown 正文。

- 文件第一行必须是 `---`；
- 不得包含读取工具附加的行号、列号或分隔前缀；
- 不得出现以 `1|`、`2|`、`15|` 等数字和竖线开头的正文；
- 不得出现匹配 `^[0-9]+\|` 或 `^[0-9]+\|[0-9]+\|` 的行；
- 读取 Source Notes 时，只提取正文内容，不得复制工具界面中的行号；
- 不得在文件末尾写入“请确认是否执行”“接下来需要的操作”等对话内容。
