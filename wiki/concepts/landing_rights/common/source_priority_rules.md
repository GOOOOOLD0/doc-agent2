# 落地许可资料来源优先级规则

## 1. 最高优先级来源

- 目标国家官方监管机构网站；
- 官方法律法规；
- 官方申请指南；
- 官方许可表格；
- 官方公告；
- 官方费用文件。

## 2. 高优先级来源

- ITU 资料；
- 区域通信组织资料；
- 政府部门官网；
- 官方公报。

## 3. 中等优先级来源

- 律所文章；
- 咨询机构报告；
- 行业协会报告；
- 运营商公开材料。

## 4. 低优先级来源

- 新闻报道；
- 博客；
- 论坛；
- 非官方转载资料。

## 5. 使用规则

1. 关键结论必须优先基于官方来源。
2. 非官方来源只能作为补充参考。
3. 如果官方来源无法确认，应标记为“待确认”。
4. 不得将第三方解释直接作为最终法律结论。
