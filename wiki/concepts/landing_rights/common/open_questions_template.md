# 待确认问题模板

分析目标国家落地许可流程后，应整理以下待确认问题。

## 1. 监管机构问题

1. 该国负责外国卫星准入的主管机构是谁？
2. 电信服务许可和频率许可是否由同一机构管理？
3. 设备认证是否由通信监管机构负责？

## 2. 许可类型问题

1. 外国卫星系统是否需要单独的落地权？
2. 是否需要电信服务经营许可？
3. 是否需要频率使用许可？
4. 是否需要地面站或网关站许可？
5. 用户终端是否需要型号核准？

## 3. 主体资格问题

1. 是否必须设立本地公司？
2. 是否允许外国公司直接申请？
3. 是否需要本地代表或本地合作伙伴？
4. 是否存在外资持股比例限制？

## 4. 频率协调问题

1. 是否要求提交 ITU filing？
2. 是否需要与已有运营商完成频率协调？
3. 未完成协调时是否可获得次级授权？
4. 是否需要提交干扰分析报告？

## 5. 商业运营问题

1. 是否允许销售卫星容量？
2. 是否允许直接向终端用户提供服务？
3. 是否允许部署网关？
4. 是否允许用户终端自由进口和销售？
