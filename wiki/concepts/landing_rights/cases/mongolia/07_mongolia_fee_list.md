# 蒙古费用清单

## 1. 文件用途

本文件汇总蒙古卫星项目已经确认的监管收费类别和计算原则。由于缺少具体项目参数、完整现行费率表及 CRC 缴费通知，本文件不提供未经证实的具体金额。

## 2. 结论摘要

### 2.1 已确认信息

1. 频率使用费与频段、带宽、发射功率、业务类别或覆盖范围等因素有关（官方来源：[《无线电频率使用和服务收费确定规则》第 2.1-2.4 条](https://legalinfo.mn/mn/detail?lawId=16530825397721)）。
2. FSS 站的频段使用费按地球至空间方向的频段使用计算，并根据频率范围区别处理（同规则第 2.2.6 条）。
3. 卫星专项规则第 4.11 条还规定，卫星频率使用费以信道租赁合同确定的频段为基础，按 CRC 批准的费率计算（官方来源：[CRC 第 37/2022 号决议附件](https://legalinfo.mn/mn/detail?lawId=16531361657351)）。
4. 监管服务费可能覆盖许可、变更、测量、投运前检查、监测和设备认证，并与人员工时和服务耗时有关（收费规则第 3.1 条）。
5. 设备认证页面确认认证费用依 CRC 2020 年第 13 号决议附件并随评估方案变化，但现有来源没有列出可直接采用的金额（官方来源：[CRC 设备合格证书申请页面](https://crc.gov.mn/for-new-license-applicants/batalgaazuulalt-e/toxirlyn-gercilgee-batalgaazuulalt)）。

### 2.2 分析推断

项目预算至少应区分频率年度费、可能的一次性权利费、监管服务费、设备认证费以及土地/建设等非 CRC 成本。各项是否同时适用、按站点还是按系统计收，需由项目参数和 CRC 决定。

### 2.3 待确认事项

卫星网络第 9.10 项许可费、现行频率系数或费率、设备认证金额、站点相关费用和续期费用，尚不能从现有来源形成可靠项目报价。

## 3. 已确认的费用类别

| 费用类别 | 已确认的计算或触发因素 | 是否有可用金额 | 官方来源 |
| --- | --- | --- | --- |
| 频率特别许可年度费 | 频段使用、发射功率、业务类别/覆盖范围 | 无 | 收费规则第 2.1 条 |
| 频段使用费 | 频率范围和带宽 | 无 | 收费规则第 2.2 条 |
| FSS 频段使用费 | 地球至空间方向；按频率范围区别 | 无 | 收费规则第 2.2.6 条 |
| 发射功率费 | 按发射机及其功率等因素 | 无 | 收费规则第 2.3 条 |
| 一次性权利费 | 新取得频率特别许可时可能适用，由 CRC 确定 | 无 | 收费规则第 2.2.8 条 |
| 监管服务费 | 许可/变更、测量、验收前检查、监测、认证；与工时和耗时有关 | 无 | 收费规则第 3.1 条 |
| 设备认证费 | 依评估方案及 CRC 第 13/2020 号决议附件 | 无 | CRC 设备合格证书页面 |
| 第 9.10 项许可相关费用 | 现有来源未明确金额或完整收费结构 | 无 | 待获取遴选文件和 CRC 缴费通知 |

## 4. 首年频率费与权利费

收费规则第 2.2.7 条规定，频段通过遴选、拍卖或权利费方式授予时，免收频率特别许可首年费用；第 2.2.8 条将新取得频率特别许可时的一次性费用定义为权利费，并由 CRC 确定。

该规则不得作以下扩大解释：

1. 第 9.10 项卫星网络许可通过遴选，就自动意味着任何相关频率许可均免首年费；
2. 首年费豁免意味着没有一次性权利费；
3. 所有卫星项目均适用该豁免。

是否适用取决于频段本身的授予方式及 CRC 缴费决定，应逐项目确认。

## 5. 形成报价所需参数

正式询价至少需要：

1. 业务类型（FSS、MSS、BSS、NTN 等）；
2. 上行和下行频率范围；
3. 申请带宽；
4. 发射机数量和输出功率；
5. 覆盖范围和站点数量；
6. 频率授予方式；
7. 卫星信道租赁合同中的频段；
8. 设备型号数量、测试资料和认证方案；
9. 是否需要投运前测量、检查或监测服务；
10. 土地、建设和其他批准范围。

## 6. 有效期、续期和付款节点

频率特别许可有效期五年；获批主体收到决定后十个工作日内支付首年费用并签约，未支付年费不发许可；付款后 CRC 在五个工作日内签发许可（官方来源：[《无线电波法》第 11.12、12.1 条](https://legalinfo.mn/mn/detail/443)）。这些是付款和发证节点，不是完整审批周期。

第 9.10 项卫星网络许可同样为五年期，但现有资料没有确认其申请费、年度费或续期费金额（官方来源：[《通信法》第 12.1 条](https://legalinfo.mn/mn/detail/523)）。

## 7. 与巴西案例的关系

本文件只复用巴西费用模块的分类方法。蒙古没有在当前来源中确认巴西式外国卫星开发权固定费用、SCM 费用或 TFI/TFF 结构，任何巴西金额均不得用于蒙古预算。

## 8. 待确认问题和下一步

1. 获取 CRC 当前生效的完整频率费率表和计算系数；
2. 获取第 9.10 项下一轮遴选文件中的费用、保证金或权利费安排；
3. 获取 CRC 第 13/2020 号决议现行认证收费附件；
4. 以具体频段、带宽、功率、站点和设备清单向 CRC 申请书面测算；
5. 确认首年频率费豁免与一次性权利费的项目适用关系；
6. 另行核算土地、建设、进口、测试和本地顾问等非监管成本。
