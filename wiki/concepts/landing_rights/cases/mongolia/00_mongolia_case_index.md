# 蒙古落地许可案例索引

## 1. 案例用途

本目录汇总外国卫星系统在蒙古开展卫星通信业务时可能涉及的市场准入、服务经营、频率、设备和地球站合规要求。正式结论以蒙古通信监管委员会（CRC）、Legalinfo.mn 所载现行蒙古文法规以及相应官方申请页面为依据。

巴西案例仅用于文件结构和问题清单参考，不构成蒙古法律依据，也不应据此推定蒙古存在巴西式外国卫星开发权或 SBH/SBS 双主体制度。

## 2. 当前结论摘要

### 2.1 已确认信息

1. CRC 是第 9.10 项“建设卫星通信网络并从事其运营和服务”特别许可及第 9.8 项频率/频段使用特别许可的主管机关（官方来源：《许可法》第 8.1 条第 9.8、9.10 项；《通信法》第 12.1、14.3 条）。
2. 第 9.10 项许可向法人授予，有效期五年，并通过至少每年一次的遴选程序授予（官方来源：《通信法》第 12.1、14.3 条）。
3. CRC 第 37/2022 号决议附件第 4.4 条原则上要求服务经营者同时持有频率许可和卫星通信网络特别许可；MSS、NTN 另有第 4.5、4.10 条专门路径，但其边界仍需 CRC 解释。
4. 地球站收发设备和用户设备按型号涉及合格认证，地球站还涉及站址、土地及其他批准（官方来源：CRC 第 37/2022 号决议附件第 4.7、4.9 条）。
5. 当前 13 个官方来源均已保存快照并生成 source note，足以形成第一版正式案例文件。

### 2.2 分析推断

蒙古的功能性市场准入路径可初步概括为“业务分类与主体资格确认 + 卫星网络特别许可 + 频率许可 + 设备认证 + 地球站/协调事项”，但不同业务形态可能适用不同组合，不能把这一概括当作统一法定流程。

### 2.3 待确认事项

1. 是否存在名称和法律效果均独立于第 9.10 项许可的外国卫星 landing rights；
2. 外国公司分支机构能否满足频率持证主体资格；
3. MSS 第 4.5 条是否以及在何种范围内排除第 9.10 项许可；
4. NTN 第 4.10 条下外国卫星运营商、蒙古地面网络持牌人和终端用户之间的商业及许可安排；
5. 下一轮遴选公告、实际申请入口、项目总周期和现行费率。

## 5. 使用边界

1. 所有关键法律结论均应回到官方网页、现行蒙古文条文及本地快照复核；Legalinfo.mn 的旧英文译文不得优先于现行蒙古文合并文本。
2. “依蒙古法律设立并运营的法人”是频率主体资格条文，不等于已确认必须设立蒙古子公司；分支机构资格仍待确认。
3. MSS 第 4.5 条不得写成已确认完全豁免卫星网络许可。
4. NTN 第 4.10 条不得写成外国卫星运营商可以独立开展 D2D 零售。
5. 一般法定审查节点、公告期、签约期和发证期不得简单相加并表述为完整项目审批周期。
6. 缺少项目参数、现行费率表和 CRC 缴费通知时，不得给出具体费用金额。
