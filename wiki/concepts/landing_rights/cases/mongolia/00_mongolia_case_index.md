# 蒙古落地许可案例索引

## 1. 文件用途

本目录用于整理蒙古卫星落地许可试点结果。当前阶段重点不是写完整法律意见，而是把已确认的官方来源映射成 agent 可回答的流程结构。

## 2. 当前结论

蒙古第一版试点已确认：

1. CRC 是通信许可、无线电频率、卫星通信网络/服务许可和设备合格认证的核心监管机构；
2. 已找到 7 个可访问、可抽取、可月度监控的官方来源；
3. 已找到 3 部 legalinfo.mn 基础法律页面，但因页面较大暂不启用月度监控；
4. CRC 频率页面中有卫星通信相关 PDF、DOCX、XLSX 附件，但项目尚未支持附件抽取，因此暂不直接监控附件。

## 5. 使用边界

Agent 回答蒙古问题时，应明确说明：

1. 当前已确认的是 CRC 官方许可和频率/认证页面；
2. 尚未确认蒙古是否存在类似巴西“外国卫星开发权”的独立许可名称；
3. 不应直接套用巴西 SBH/SBS 双主体结构；
4. 费用、审批周期、本地实体要求和附件材料需要继续核验。
