# 巴西案例索引

## 1. 案例名称

Spacesail 在巴西开展卫星业务的落地许可流程。

## 2. 案例用途

本案例用于作为其他国家卫星落地许可分析的参考样板。

当后续分析泰国、蒙古、印尼、墨西哥等国家的卫星落地许可流程时，应参考本案例的模块划分，但不得直接假设目标国家的许可体系与巴西相同。

## 3. 巴西案例核心结论

Spacesail 在巴西开展卫星通信业务时，需要重点关注以下事项：

1. 外国卫星开发权 / 落地权；
2. 面向终端用户的电信服务授权；
3. 频率协调；
4. 设备认证和批准；
5. 地面站、网关站和空间站许可；
6. 相关费用；
7. 适用法规。

## 5. 使用方式

Agent 在分析其他国家卫星落地许可时，应：

1. 先读取 common 文件夹中的国家落地许可分析 SOP；
2. 再读取本巴西案例索引；
3. 根据任务需要读取具体模块文件；
4. 将巴西流程作为检查清单和对比基准；
5. 对目标国家进行开放式检索；
6. 标记无法确认的信息为“待确认”。
